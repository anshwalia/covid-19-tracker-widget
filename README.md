# COVID-19 Tracker Widget
A simple widget to track Corona Virus (COVID19) spread across India.

## Tech Stack
* ElectronJS
* NodeJS
* Bootstrap 4
* Library
  * JQuery
  * axios
  * cheerio
  * chart.js

## Getting Started
#### Clone the repository.
#### Install NodeJS
#### Open terminal in project directory.
#### Install all the required modules
```
npm install
```
#### Execution
```
npm start
```
