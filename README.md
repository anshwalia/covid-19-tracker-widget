# COVID-19 TRACKER WIDGET

A simple widget to track Corona Virus (COVID-19) spread across India.

## Tech Stack
* ElectronJS
* NodeJS
* Bootstrap 4
* Library
  * JQuery
  * axios
  * cheerio
  * chart.js
  * anime.js

## Getting Started
#### Clone the repository.
#### Install NodeJS
#### Choose Version 1,2
#### Open terminal in version directory.
#### Install all the required modules
```
npm install
```
#### Execution
```
npm start
```
